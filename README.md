# prettier-config

Standard prettier configuration by STRG.

## Installation

```bash
# install with npm
npm install -D @strg/prettier-config

# or with yarn
yarn add -D @strg/prettier-config
```

## Usage

```js
// .prettierrc.js
module.exports = {
  ...require('@strg/prettier-config')
}
```
